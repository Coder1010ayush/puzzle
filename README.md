# Sliding Number Puzzle

## Description
This is a simple sliding number puzzle game implemented in JavaScript. The game features a 3x3 grid with numbers from 1 to 8 and one empty space. The goal is to arrange the numbers in order by sliding them into the empty space.

## How to Play
1. Click on any number adjacent to the empty space to slide it into that space.
2. Your aim is to arrange the numbers from 1 to 8 with the empty space at the bottom-right corner.

## Setup Instructions
1. Clone the repository or download the index.html file.
2. Open the index.html file in your web browser.
